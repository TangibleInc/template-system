## Template

Template language defined by dynamic tags

- Core tags
- Core logic rules
- HTML module
- CodeMirror code editor
- Integration with Advanced Custom Fields
- Modules

  Many of these integrate with libraries in the Interface module.

  - Calendar (idea)
  - Chart (beta)
  - Embed
  - Form (alpha)
  - Glider
  - Markdown
  - Math
  - Prism
  - Sass
  - Slider
  - Table
